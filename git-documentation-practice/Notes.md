# Git Beginner Lab Notes

## Objective
Practice using Git and GitHub to document and upload a lab project.

## Environment
- macOS
- VS Code
- Git installed locally
- GitHub account

## Commands Used

git status
git add .
git commit -m "message"
git push

## Issues Encountered

- Password authentication failed when pushing to GitHub
- Learned that GitHub requires a Personal Access Token instead of password

## Solutions

- Generated a Personal Access Token
- Used the token as password when pushing

## Key Takeaways

- Saving in VS Code does NOT update GitHub
- Must run add → commit → push
- Always document errors and how they were resolved
